# Standards

Pasta oficial de padroes obrigatorios do Systex AI Engineering Framework.

## Estrutura

```text
design-system/ -> Design System Systex, tokens, UI, componentes e White Label
engineering/   -> padroes tecnicos por stack, API, banco e code review
operations/    -> deploy, hospedagem, backup e observabilidade
```

## Regra

Todo projeto deve declarar quais padroes se aplicam antes de iniciar implementacao. Excecoes devem ser registradas com motivo, impacto e risco residual.
