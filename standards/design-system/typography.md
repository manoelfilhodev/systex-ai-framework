# Tipografia

## Diretriz

A tipografia Systex deve favorecer leitura, hierarquia clara e uso profissional em interfaces densas, dashboards e fluxos operacionais.

## Regras

- Usar no maximo duas familias tipograficas por produto.
- Definir escala tipografica centralizada.
- Evitar fontes decorativas em sistemas operacionais e administrativos.
- Preservar contraste entre texto principal, secundario e informacao auxiliar.
- Nao usar texto muito pequeno para dados criticos.
- Manter hierarquia consistente entre titulos, subtitulos, labels, tabelas e botoes.

## Escala recomendada

```text
display: 40px / 48px
h1: 32px / 40px
h2: 24px / 32px
h3: 20px / 28px
body: 16px / 24px
small: 14px / 20px
caption: 12px / 16px
```

## Aplicacao

Interfaces enterprise devem priorizar clareza e densidade controlada, nao impacto visual gratuito.
