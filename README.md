# 🧠 Systex AI Engineering Framework

Framework de engenharia orientado por IA para desenvolvimento de software com padronização, qualidade e escalabilidade.

Este repositório define um sistema de agentes especializados que trabalham de forma orquestrada para transformar ideias em sistemas prontos para produção.

---

## 🎯 Objetivo

Garantir que todos os projetos da Systex sigam um padrão consistente de:

- Arquitetura
- Qualidade
- Segurança
- Organização
- Escalabilidade

---

## 🧩 Estrutura do Framework

agents/     → definição dos agentes  
atlas/      → orquestração (ATLAS)  
prompts/    → comandos prontos  
templates/  → templates de handoff e fluxo  
docs/       → documentação gerada por projeto  

---

## 🧠 Agentes Systex

| Agente | Função |
|------|--------|
| 👑 ATLAS | Orquestração e direção do projeto |
| 🧠 ATHENA | Regras de negócio |
| 🏗️ PROMETEU | Arquitetura |
| 🗄️ GAIA | Banco de dados |
| ⚙️ VULCAN | Estrutura base |
| 🔴 ARES | Backend/API |
| 🎨 APOLLO | Frontend/Admin |
| 📱 HERMES | Mobile/Flutter |
| 🧪 ORION | Testes |
| 🛡️ HADES | Segurança |

---

## 🚀 Fluxo Oficial de Inicialização de Projetos

Siga este fluxo em todo novo projeto.

1. ATLAS → Orquestração  
2. ATHENA → Regras de negócio  
3. PROMETEU → Arquitetura  
4. GAIA → Banco de dados  
5. VULCAN → Estrutura base  
6. PROMETEU → API  
7. Início do código (ARES / APOLLO / HERMES)  
8. Validação (ORION / HADES)

---

## ⚠️ Regra crítica

Não iniciar código antes de:
- regras de negócio
- arquitetura
- banco de dados
- contratos de API

---

## 🧠 Como usar

1. Criar projeto usando este repositório como template  
2. Abrir no Codex  
3. Executar agentes na ordem correta  
4. Gerar documentação  
5. Só depois iniciar código
